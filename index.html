BookFinder

BookFinder is a Python automation script that searches multiple bookstores, including Blackwells (UK) and Finlandiakirja (Finland), and compiles the top 10 relevant listings from each site.

The program extracts the book title, price, and link from each result and exports the data into both CSV and TXT formats for easy viewing.

Pokémon Info Fetcher

Pokémon Info Fetcher is a Python script that retrieves detailed information about any Pokémon using the PokeAPI. It fetches types, stats, abilities (with descriptions), and calculates weaknesses and resistances (×4, ×2, ×0.5, ×0).

The program outputs the results into a text file and automatically opens it, making it easy to view and save. A standalone executable is included for users who don’t have Python installed.
